Python (data analysis and plotting, free, open source)
- We use mainly numpy, pandas, and SciPy packages for data analysis, Matplotlib and Seaborn packages for data plotting;
- Coding should be done in JupyterLab package for better sharing and reading, and we suggest JupyterLab Desktop;
- Read this to know why we use Python, not Originlab or Excel;
- For introductive tutorials, we suggest Scientific Computing for Chemists with Python by Charles J. Weiss, and Python for Data Analysis from MolSSI.
