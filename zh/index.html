郑课题组 / The Zheng Group

电化学界面与转化实验室 @ GTIIT

郑课题组由郑蔚然博士负责，研究电化学界面在工作状态下的结构演变、光谱响应、电化学响应和失活过程。我们结合原位/工况（operando）表征、电催化稳定性研究、定量电化学方法和 Python 数据工作流，理解催化活性、选择性和稳定性如何随时间演化。

研究目标 把可靠实验、合理解释和可复现分析连接起来

Research

研究方向

我们关注电化学界面在真实工作条件下的变化，并尽量避免测量和解释中的常见误区。

原位/工况（operando）光谱电化学与 EC-AFM

在反应进行时跟踪界面的形貌、光谱和产物变化，把观察结果与活性、选择性和稳定性联系起来。

DEMS UV-vis Raman EC-AFM

电催化剂稳定性与失活

解析长期测试中活性、选择性、阻抗参数、电化学活性面积（ECSA）和界面结构的变化，区分材料或界面失活与实验伪影。

稳定性测试 失活分析 tr-EIA

定量电化学方法与数据工作流

围绕欧姆降（iR）校正、电化学阻抗谱（EIS）分析、数据标准化和 Python 工具，建立更透明、更可复现的电化学解释流程。

iR 校正 EIS Python
