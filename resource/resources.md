## Software Recommendations

### Python

URL: https://www.python.org/
Note: Free
Featured: true

Essential programming language for data processing, plotting, system simulation, and hardware control in electrochemistry research.

- **Core Packages:** NumPy, Pandas, SciPy, Matplotlib, Seaborn for scientific computing and visualization.
- **Recommended Environment:** [Visual Studio Code](https://code.visualstudio.com/) with [Codex](https://openai.com/codex/) for AI-assisted coding, data analysis, and documentation.
- **Why Python:** Read [this article](https://iopscience.iop.org/article/10.1149/2754-2734/acff0b/meta) explaining our preference for Python over OriginLab or Excel.
- **Learning Resource:** View Dr. Zheng's ECS Webinar on Python for electrochemistry [here](https://physicsworld.com/a/introducing-python-for-electrochemistry-research/).

### Blender

URL: https://www.blender.org/
Note: Free

Professional 3D modeling, animation, rendering, and visual effects software for creating scientific visualizations.

### Fiji (ImageJ)

URL: https://fiji.sc/
Note: Free

Open-source image processing software specialized for scientific image analysis, particularly useful for electron microscopy images.

### VESTA

URL: https://jp-minerals.org/vesta/en/
Note: Free

3D visualization software for crystal structures, volumetric data, and crystal morphologies in materials science.

### Gwyddion

URL: https://gwyddion.net/
Note: Free

Modular software for scanning probe microscopy (SPM) data analysis and visualization.

## Lab-Developed Tools

### VersaParse

URL: https://github.com/zhenggroup/versaparse
Note: Free, Open Source

Python package for parsing and processing .PAR files from electrochemical instruments.

- **Purpose:** Automated parsing of PAR electrochemical data files for integration into Python workflows.
- **Installation:** Available on [GitHub](https://github.com/zhenggroup/versaparse).
