Software

Python(Free)

Used for data processing, plotting, system simulation, and hardware control.

  • Packages: NumPy, Pandas, SciPy, Matplotlib, Seaborn.
  • Environment: JupyterLab Desktop for better sharing and reading.
  • Read this article to know why we use Python over OriginLab or Excel.
  • View Dr. Zheng's ECS Webinar on Python for electrochemistry here.

Blender(Free)

Software for 3D modeling, animation, rendering, and visual effects.

Fiji (ImageJ)(Free)

Open-source image processing for scientific image analysis (e.g., for EM images).

VESTA(Free)

3D visualization for crystal structures, volumetric data, and crystal morphologies.

Gwyddion(Free)

A modular program for scanning probe microscopy (SPM) data analysis.
