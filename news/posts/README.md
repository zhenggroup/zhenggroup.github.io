# Markdown News Posts

Add new `.md` files in this folder, then list the filename in `manifest.js`.

Use this front matter format:

```md
---
id: short-url-anchor
date: 2026-05-10
title: Your News Title
imageSrc: ../img/example.jpg
imageAlt: Short image description
imageFloat: left
imageWidth: 30%
imageCaption: Optional caption
---

Write the news body in Markdown here.
```

Only `date` and `title` are required. Leave image fields blank or remove them when there is no image.
